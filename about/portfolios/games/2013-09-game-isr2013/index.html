ISR 2013: Burning Civilization
團隊開發

Lua Love

這是參加 Indie Speed Run 2013 的作品，是以 Burning Civilization 為遊戲名稱、Camp Fire 為遊戲元素去製作的小型遊戲。這個專案是和 Raincole、Davy、mrjoshuachao 組成 Fludot 團隊去共同製作，在 30 小時內的時間進行黑客松，從抽題、發想、開發到完成，最後於活動尾聲時進行發表。

這個專案是我第一次參加遊戲獨立製作的活動，也是第一次參加跨夜黑客松，要在短時間內針對抽到的題目進行遊戲製作是一項很有趣的挑戰，也在過程中學習如何團隊合作。從腦力激盪、繪製原型、分工開發，到後期的不斷測試、修正，都仰賴團隊之間彼此互相的幫忙，以及持續不斷的溝通與討論，這是一個很實用的經驗。

在這個專案裡，mrjoshuachao 負責美術設計。我則和 Raincole、Davy 共同負責程式開發。在這次開發中，我從夥伴身上學到了如何開發一個遊戲的基礎知識，暸解到遊戲背後是如何運作的，以及該怎麼設計整個遊戲專案的架構、要有哪些類別與方法、要怎麼繪製與更新畫面等。從到完成一項作品，讓我感受到自己踏出了遊戲程式開發的第一步。
